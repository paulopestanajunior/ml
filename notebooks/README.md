# Notebooks do Plano 2

Este diretório é reservado para notebooks Jupyter usados nas atividades práticas de cada semana. Sempre que uma tarefa solicitar a implementação de algoritmos, testes de modelos, simulações ou criação de pipelines, crie um notebook aqui com um nome descritivo (por exemplo, `semana06_projeto_multiagentes.ipynb`).

Recomendações:

* Use o ambiente do **Google Colab** ou Jupyter Lab local para desenvolver os notebooks. Se utilizar o Colab, lembre‑se de salvar uma cópia no seu drive e exportar como `.ipynb` para versionamento.
* Documente bem cada célula com comentários e explicações para facilitar a revisão futura.
* Armazene datasets grandes na pasta `datasets/` e faça a leitura no notebook com caminhos relativos.
* Ao finalizar um notebook, marque a tarefa correspondente como concluída no checklist da semana.
