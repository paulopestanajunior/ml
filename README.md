# Plano 2 — Machine Learning, LLMs e Sistemas Multiagentes

Plano de estudos completo para evoluir de **Machine Learning aplicado** até **LLMs, RAG, sistemas multiagentes, observabilidade e produção**.

## Objetivo

Entender:

- como problemas de negócio viram problemas de modelagem
- como escolher, treinar e avaliar modelos de ML
- como funcionam Transformers e LLMs
- como construir RAGs de verdade
- quando usar agentes e quando **não** usar
- como avaliar, observar e colocar sistemas agentic em produção
- como julgar viabilidade de produto, custo, risco e governança

## Ritmo sugerido

- 8 semanas
- 3h por dia
- 6 dias por semana
- 1 dia de revisão leve / atraso
