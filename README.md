# Plano 2 — Machine Learning Avançado, LLMs e Sistemas Multiagentes

Este plano aprofunda fundamentos de **Machine Learning**, avança para **LLMs**, **RAG** e **sistemas multiagentes**, e fecha com **avaliação**, **observabilidade** e **produção**.

## Visão geral

- duração sugerida: 8 semanas
- carga diária: 3h por dia
- foco: teoria + prática + critérios de produto
- objetivo final: entender quando um produto com LLM/multiagentes faz sentido e como colocá-lo em produção
