# Resumos e Fichamentos

Utilize esta pasta para armazenar resumos, fichamentos e reflexões das leituras feitas em cada semana. Criar resumos é uma ótima maneira de fixar conceitos e servir de referência rápida no futuro. Sugestões de uso:

* Crie um arquivo `semanaXX-resumo.md` com uma síntese dos capítulos lidos, principais definições, algoritmos e insights extraídos das atividades práticas.
* Inclua citações curtas e páginas/líneas de referência quando relevante. Lembre‑se de que trechos longos de textos protegidos devem ser parafraseados.
* Anote dificuldades encontradas e dúvidas para discutir em grupos de estudo ou pesquisar posteriormente.
* Use tabelas comparativas ou bullet points para organizar melhor a informação.
